Welcome to the LA data project on an immigration workshop. This website is used to show the information on the events that was taken from past events throughout various of immigrate workshop. You may want to use this website on immigration workshops to get information like date of events, location, organization, contacts, languages and notes provided from the organization. In order to get this information, you must insert a zip code on the line depending on where you are located at or a choosing for your choice. Afterwards, click that like button ;). Be aware that a few zip codes doesn't work either because the information isn't provided or an error occur. NOTE: If you want a zip code example, you may insert 90032 to test how our website works.
